PROJET TKT Ionic (ZooPlanner)

CONTEXTE

Projet de classe réalisé par groupe de 3 en 16 heures. Grâce à un cahier des charges préalablement établi, nous avons dû développer une application web en utilisant la librairie React. Pour communiquer avec une base de données MySQL, nous avons également mis en place une API-REST via un serveur NodeJS. Enfin, nous avons mis en production cette Web App pour qu'elle soit accessible aux utilisateurs
